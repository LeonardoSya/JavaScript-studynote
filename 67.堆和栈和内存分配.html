简单数据类型(值类型、基本数据类型) 在存储时变量中存储的是值本身 string number boolean undefined null 注意typeof null 返回值是object(这个比较特殊) 所以当new一个对象但暂时没想好放啥时 就赋值null 复杂数据类型(引用类型) 在存储变量中存储的仅仅是地址(引用) 通过new关键字创建的对象(系统对象、自定义对象)如Object Array Date等 都属于引用类型 栈(操作系统): 由操作系统自动分配释放存放函数的参数值、局部变量的值等 基本数据类型放到栈里面 其操作方式类似数据结构中的栈 堆(操作系统): 存储复杂类型(对象) 一般由程序员分配释放 若程序员不释放 由垃圾回收机制回收 简单数据类型 放到栈里面(存放的是值) 复杂数据类型 首先在栈里面存放一个地址 真正的对象实例存放在堆空间中
